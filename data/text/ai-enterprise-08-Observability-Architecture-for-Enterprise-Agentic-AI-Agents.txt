===== PAGE 5 =====
Traditional Application vs AI Agent
DETERMINISTIC · Traditional app
●
Fixed control flow, known endpoints
●
Errors carry explicit codes
●
Latency is the main SLO
●
Schema-stable inputs & outputs
PROBABILISTIC · Agentic system
●
LLM picks the next step at runtime
●
Same input ≠ same trajectory
●
Failures are semantic, not status-code
●
Quality + cost + safety are all SLOs
●
Free-text in, free-text + tool calls out
Observability for agents must capture reasoning, tool use, retrieval, and intent
