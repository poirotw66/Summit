===== PAGE 27 =====
FAQ︓常見問題解答
Q&A — 六個開發者最常問的問題
Q1  這個專案最適合學什麼？
production-oriented Agent 架構：⼯具治理、狀態管理、串流 UI、安全稽核、
評測與部署的完整骨架。
Q2  為什麼不讓模型直接查資料庫？
MCP Toolbox 讓查詢被⼯程定義與約束，避免模型⾃由產⽣ SQL，並把業務邏
輯（budget_fit）前置到⼯具層。
Q3  Session memory 和聊天歷史有什麼不同？
聊天歷史是非結構化上下⽂；session memory 是可檢查、可更新、可測試的結
構化狀態，例如年齡、預算與上次推薦商品。
Q4  SSE 和 WebSocket 各⾃負責什麼？
SSE ⽤於⼀般⽂字 Agent run 的單向串流；WebSocket ⽤於 Live mode 的全雙
⼯⾳訊、影像與⽂字互動。
Q5  如何避免 Agent 洩漏敏感資料？
JWT 驗證、PII redaction、public state filtering、audit log payload 
redaction，以及 safety evalsets 驗證。
Q6  這個架構可以換成其他業務領域嗎？
可以。保留 ADK + toolset + state + eval + observability 骨架，替換 schema、
⼯具、prompt 與 evalsets 即可移植。
